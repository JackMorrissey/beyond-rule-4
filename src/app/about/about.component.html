About

This application is an open source side-project of a software developer chasing financial freedom. It was created with the intent of providing personal motivation to become debt free and to continue well beyond that goal.

The announcement that YNAB was having an API contest gave me the motivation (and the ever needed deadline) to make this available for an audience bigger than myself. If you have any questions or comments, feel free to send me an email at beyondrule4@jmmorrissey.com.

This application doesn't collect or store any personal data before or after authorizing with YNAB. We may use browser cookies or browser local/session storage for application functionality and analytics but nothing nefarious.

When we authorize with YNAB, we request read-only access. We do not add or edit any information to your account or budget.

To revoke authorization at any time, visit https://app.youneedabudget.com/settings, scroll to "Authorized Applications", and click "Revoke" for Beyond Rule 4. Thanks for trying it out!

This application is not sponsored, endorsed, or supported by YNAB.

YNAB OAuth

This application follows the Terms of Service for the YNAB API

  • The data obtained through the YNAB API is not stored on the application server. We may use browser local/session storage and/or cookies for application functionality, but API data is only received and used by the client browser.
  • Data obtained through the YNAB API will not unknowingly be passed to any third-party.
  • The application will never ask for any financial account credentials.
